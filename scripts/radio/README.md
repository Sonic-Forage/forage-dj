# Forage Radio - PowerShell Version

This is a native Windows PowerShell radio player for tracks generated by your ComfyUI (ACE-Step 1.5 + Stable Audio 3).

## How to Launch

**Best method:**
Double-click `Launch-Forage-Radio.bat`

**Alternative (PowerShell):**
```powershell
cd Z:\IMF2045\forage-dj\scripts\radio
.\Forage-Radio.ps1
```

## Controls (while running)

- **Space** → Play / Pause
- **N** → Next track
- **P** → Previous track  
- **R** → Refresh playlist (picks up newly generated tracks)
- **Q** → Quit

## Important: Window Closes Immediately?

If the window opens and closes right away:

1. **Use the .bat file** (it has `-NoExit`)
2. Or run it from an **already open PowerShell window** instead of double-clicking.

The script now has much better error handling and will try to keep the window open.

## Requirements

- `ffplay.exe` (part of ffmpeg) must be available.
  - The script tries several common locations.
  - If it can't find it, it will tell you clearly.

## How It Works

- Scans `Z:\C0MFY\ComfyUI\output\audio\` for .mp3 files.
- Plays them using ffplay (headless).
- Auto-advances to the next track when one finishes (real radio behavior).
- Press **R** regularly while generating new tracks in ComfyUI — they will appear in the playlist.

## Tips

- Keep this window open in the background while you generate in ComfyUI.
- Hit **R** after a few tracks finish rendering to see them show up.
- The playlist shows newest tracks first.

This gives you a simple but functional terminal radio experience while you keep creating new music with your ACE-Step and Stable Audio workflows.
