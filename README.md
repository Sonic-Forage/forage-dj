# forage-dj
Sonic Forage DJ - Open-source AI-powered autonomous DJ interface/DAW. Prompt-to-track generation with Stable Audio 3 (small-music, small-sfx, medium), Numark NS7II MIDI mapping, real-time mixing, auto-mix agent, voice commands. Built on the Sonic Forage mycelium ecosystem for rave culture, safety gates, and creative AI agents.
