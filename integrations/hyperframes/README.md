# HyperFrames Integration (HeyGen)

**Repo**: https://github.com/heygen-com/hyperframes

## What is it?
HyperFrames is HeyGen's platform for creating **interactive AI video experiences** — talking video avatars/characters that can have conversations, explain concepts, tell stories, or act as promotional hosts.

Perfect for Sonic Forage because:
- We can create an interactive "Rex" character (Grok TTS voice) that explains what the fuck we're building in a fun, memorable way.
- Great for events, workshops, streams, and onboarding new people.
- Ties beautifully with our existing Avaturn reactive characters work.

## Vision for Sonic Forage

**"Rex" — The Official Sonic Forage AI Host**

- Uses Grok TTS voice (fun, slightly unhinged, rave-aware).
- Can explain:
  - What the Music Diffusion OS actually is
  - How the workstation + swarm distribution works
  - Why we do harm reduction donut raves
  - How to run your own autonomous set
  - The philosophy behind seed bombing and giving music away
- Interactive: People at events or on streams can "talk" to Rex.
- Fun promotional tool that feels on-brand instead of corporate.

## How to Use

1. Go to HeyGen HyperFrames and create a project.
2. Generate scripts using Grok (or the personalities system in this repo).
3. Use Grok TTS or HeyGen's voice cloning for "Rex".
4. Deploy as interactive video on:
   - Event websites
   - Twitch/Kick overlays or pre-roll
   - Workshop materials
   - The retro OS as an "explain" button (future)

## Integration Ideas with Existing Systems

- **Personalities**: Feed a DJPersonality (e.g. Neon or Flux) into the script so Rex can roleplay different vibes.
- **Avaturn**: Combine HyperFrames talking head with our reactive 3D characters.
- **Swarm Distribution**: Rex can narrate seed bomb releases or new library drops.
- **Rave Prep**: Rex can give the actual harm reduction + PLUR speech in video form before events.
