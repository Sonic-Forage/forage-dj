# Sonic Forage / forage-dj Branding Assets

This folder contains the official badass logos generated for the project.

## Files

- **sonic_forage_main_logo.jpg** — Primary logo with text (best for most uses)
- **sonic_forage_icon_only.jpg** — Clean icon/emblem version (great for app icons, favicons, merch)
- **sonic_forage_icon_monochrome.jpg** — Black & white version (perfect for laser engraving, patches, stickers, print)
- **sonic_forage_banner_wide.jpg** — Wide banner/header version (website hero, GitHub social preview, event posters, etc.)

## Usage Guidelines

- Use the main logo for general branding.
- Use the icon versions when space is limited.
- The monochrome version works well on dark or light backgrounds.
- The wide banner is ideal for headers and wide formats.

## Generation Notes

These were created using the xAI Imagine model with prompts focused on:
- Cyberpunk + organic mycelium + strawberry DJ hybrid
- Rebellious rave energy
- Professional yet powerful logo design
- Dark moody palette with neon pink/purple/green accents

Feel free to generate variations later (different colorways, more minimal, more aggressive, etc.).
